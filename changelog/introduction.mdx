## Apr 29, 2026

### Feature Updates
Features;

Added new filtering UI for tables, which allows more filters to be included without taking up space on the page. Further, searching for items in the filters is much easier for all filter options.

We now display filtered items in badges/chips above the table, along with the column that is being sorted, making it easier for the user to identify the filtered rows.

Increased API performance by ~2x with some code enhancements.

### Issues Resolved

Removed M3 container inputs for palletized stock on organization settings as these are no longer needed.

## Apr 25, 2026

### Feature Updates

You can now view a record in a "new tab" from the preview sidebar, making it easier to multitask in Flip.

The browser now tracks the user's table page, ensuring they return to the same page when navigating back.

### Issues Resolved

Fixed a 404 error in the Products form when updating Format & Specifications.

Product Extra Costs and Savings are now saved only after pressing Update when editing a product. Previously, these changes were applied instantly instead of waiting for the user to press "Update".

## Apr 20, 2026

### Feature Updates

**Edit Extra Cost and Savings library items from Formats page**

You can now edit Extra Costs and Savings from the Formats page. Previously, you had to go to the Settings page to edit Extra Costs and Savings in the library.

<Note>
  Be careful when updating the library from the Formats page. This will affect any Formats where this Extra Cost or Saving is used.
</Note>

**Double-Click to View Records**

You can now set your usability settings to display records when double-clicking a row in a table. Previously, you had to click the row and then select "View" from the preview sidebar.

To turn this feature on, switch on "Enable double-click to open detail view" from your Profile settings.
